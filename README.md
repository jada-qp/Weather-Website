# Fullstack Minimal Scaffold

## Run Both

```
npm install
npm run install:all
npm run dev
```

## WeatherAPI

Create `backend/.env` with your WeatherAPI key:

```
WEATHERAPI_KEY=your_key_here
```

## Frontend

```
cd frontend
npm install
npm run dev
```

## Backend

```
cd backend
npm install
npm run dev
```

The frontend expects the backend at `http://localhost:3001`. Adjust `frontend/.env.example` as needed.
